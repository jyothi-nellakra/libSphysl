# libSphysl: The Simple Physics Simulation Library

Copyright (C) 2026 Jyothiraditya (Jyothi) Nellakra

This project houses a series of Physics simulations that I was working on during undergrad. You can build and run the demos with `make -j8 demos` in either the project's root folder or in the folder `old/` which houses more demos before I tried to refactor the codebase.
